## Fase 0 (2026-2028): Validación Piloto

**Presupuesto:** €9M
**Potencia:** 5 MW
**Duración:** 22 meses

**Objetivos:**
- Confirmar viabilidad física del sistema
- Validar estimaciones de OPEX/CAPEX
- Generar datos para decisión de escala

**Hitos clave:**
- Mes 4: Permisos iniciales obtenidos
- Mes 10: Fabricación componentes completada
- Mes 16: Instalación submarina completada
- Mes 22: 6 meses de datos operacionales
- Mes 28: Informe Go/No-Go definitivo

---

## Fase 1 (2028-2030): Piloto Comercial 26 MW

**Presupuesto:** €89M (si Go en Fase 0)
**Potencia:** 26.5 MW + OI 100k m³/día
**Duración:** 36 meses

**Objetivos:**
- Construir primera planta a escala completa con desalinización modular
- Validar LCOE real (€45/MWh objetivo con agua)
- Conectar a red eléctrica española
- Negociar PPA a 10-15 años + contrato suministro agua con Junta Andalucía

**Financiación:**
- 40% equity (promotores)
- 60% deuda bancaria (BEI/ICO Energy)

## Fase 2 (2030-2032): Segunda Unidad

**Presupuesto:** €60-80M
**Potencia adicional:** 26 MW (total: 52 MW)
**Ubicación:** Segunda localización costa Huelva

**Ventajas de segunda unidad:**
- Curva de aprendizaje (coste -15-20%)
- Infraestructura compartida (caseta control, cable tierra)
- CAPEX unitario menor

## Fase 4 (2036-2045): Madurez

**Presupuesto:** €300-500M
**Potencia:** 10-15 plantas (350-500 MW)

**Actividades:**
- Primeras plantas entran en ciclo de renovación
- Tecnología de segunda generación (potencia 50 MW/planta)
- Inicio exportación tecnología internacional

**Impacto energético España:**
```
500 MW × 95% CF × 8760h = 4,161 GWh/año
Consumo eléctrico España 2026: ~250,000 GWh/año
Contribución: 1.7% consumo total
```

## Conclusión: Una Oportunidad Real

El proyecto representa una oportunidad legítima para crear una nueva industria energética española basada en tecnología propia, con:

- **LCOE competitivo** frente a todas las alternativas marinas
- **Energía base** (95% disponibilidad) que el sistema necesita
- **Escalabilidad** a lo largo de todas las costas españolas con 500m de profundidad
- **Potencial de exportación** de tecnología española a nivel global

No es una revolución energética. Es **ingeniería bien planteada** en un nicho específico y valioso.
